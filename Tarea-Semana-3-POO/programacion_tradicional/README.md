## Descripción

Este proyecto contiene dos programas desarrollados en Python para gestionar información básica de mascotas.

### Programa Tradicional

Utiliza variables y funciones para registrar y mostrar la información de una mascota mediante entrada de datos por teclado.

### Programa Orientado a Objetos

Utiliza una clase llamada Mascota, atributos, métodos y objetos para gestionar la información de las mascotas.

## Reflexión

La Programación Tradicional organiza el código mediante variables y funciones, siendo adecuada para programas pequeños. Sin embargo, cuando el sistema crece, puede resultar más difícil administrar la información.

La Programación Orientada a Objetos permite representar entidades mediante clases y objetos, facilitando la reutilización del código, la organización de los datos y el mantenimiento del programa. Además, mejora la escalabilidad de los sistemas y permite aplicar conceptos como encapsulación, abstracción, herencia y polimorfismo.
